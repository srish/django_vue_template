=== Running the app ===

Django server:
* `git clone https://github.com/srish/django_vue_template`
* `cd django_vue_template` 
* `python manage.py runserver`

Vue app:
* `cd frontend`
* `npm install`
* `npm run serve`

Navigate to `http://127.0.0.1:8000/home/`

=== Resources ===
